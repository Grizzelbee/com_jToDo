jToDo - Eine ToDo Liste für Joomla
**********************************

Änderungen in Version 1.0.8:
Backend:
[+] Auf den verschiedenen Management-Seiten gibt es jetzt Buttons um zwischen den Managern 
    schneller umschalten zu können, weil der Weg über das Menü sehr mühselig ist.

Änderungen in Version 1.0.7:
Backend:
[!] Beim Klick auf den Link eines ToDo-Eintrages wurde dieser nicht zum editieren geöffnert, sondern ein
    neuer (leerer) Eintrag  angezeigt. Das solte so nicht sein. eintrag wird jetzt wieder zum editieren geöffnet.

Änderungen in Version 1.0.6:
Backend:
[!] Beim Klick auf das Erledigt-Status-Icon in der Todos-Listenansicht wurde der Freigegeben-Status geändert. 
    Das ist jetzt behoben und der Erledigt-Status wird korrekt umgeschaltet.

Änderungen in Version 1.0.5:
Frontend:
[!] Es hatte sich ein Fehler beim Aktualisieren der Besuchsaufzeichungen eingeschlichen. Das ist jetzt wieder behoben.

Änderungen in Version 1.0.4:
Frontend:
[+] Neu/Aktualisiert-Anzeige funktioniert jetzt
[+] Beim Wechslen des ToDo-Status werden jetzt weitere Daten, wie Updatezeitpunkt etc. korrekt gesetzt.
    
Änderungen in Version 1.0.3:
Frontend:
[+] Der Besuch einer Projektseite durch einen angemeldeten Benutzer wird protokolliert. Das ist für die Neu/Aktualisiert-Anzeige
    notwendig
[+] ToDo-Umschaltlinks dürfen nur aktiv gesetzt werden, wenn der Besucher angemeldet ist. Kein Öffentlichlicher Zugriff darauf.

Backend:
[+] Die Seiteneinstellungen können pro Menüeintrag relativ weitreichend konfiguriert werden
    -> Logo kann ausgewählt werden
    -> ToDo/Done Icon kann ausgewählt werden
    -> Seitenüberschrift kann gesetzt werden
    -> Seitenüberschrift und Logo können getrennt ein- und ausgeschaltet werden
[+] Es können eigene Grafiken für Logo / Done- und ToDo-Icon verwendet werden.   
    Dazu müssen sie nur in \media\com_jtodo\images\logos bzw. \media\com_jtodo\images\stateindicators
    abgelegt werden.
[!] Die beiden Daten "inserted" und "updated" werden jetzt korrekt in die Datenbank geschrieben.
